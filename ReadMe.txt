Описание корректных данных для работы с программой
По условию имеем графы ориентированные и неориентированные, невзвешенные, без кратных рёбер и петель

1) Ввод графа
Вершины нумеруются с единицы при вводе.
В качестве разделителя используется пробел.

– Матрица смежности
Квадратная матрица, состоящая из нулей и единиц. 
Так как рассматриваются графы без петель, то на диагонали матрицы должны быть нули

– Матрица инцидентности
Матрица, где i-я строка обозначает i-ю вершину, а j-й столбец – j-е ребро.

– Список смежности
Сначала в строке вводится номер вершины, затем вводятся вершины, смежные с ней.

– Список рёбер
Список пар вида: (начало ребра, конец ребра). 
Так как рассматриваются графы без петель, то начало и конец совпадать не могут.
Так как рассматриваются графы без кратных рёбер, то все пары уникальны.
При вводе списка рёбер рёбра нумеруются в порядке ввода.

Чтобы while(getline) можно было прервать (ввод через консоль), 
введите стоп-слово stop на следующей строке после входных данных
(сочетание клавиш Ctrl+D (Windows) не учтёт выбор вывода, и программа работать не будет)

Файловый ввод позразумевает, что в папке с решением должен быть файл input.txt. 
Выходной файл в принципе удалять можно.
Редактировать файлы можно.
Выходной файл автоматически перезаписывается при перезапуске программы.

2) Полустепенью вершины ориентированного графа считается число исходов 
(число дуг, исходящих из вершины)
